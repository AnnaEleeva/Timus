grpc - использует протокол http 2
rest - использует протокол http 1.1

Первая версия протокола http требовала дожидаться получения ответа перед отправлением следующего запроса в рамках одного соединения.
Во второй версии протокола - это исправили, соединение может использоваться без ожидания завершения уже отправленного запроса.
